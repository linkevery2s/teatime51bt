このアプリについて

「最近、頭使ってないなぁ。」

そう思うことはありませんか？

このアプリは空いた時間に少し頭の体操ができるアプリです。

Teatimeなど、ほんの僅かな時間に脳トレが出来てしまいます。

社会人の方はもちろん、お子さんから高齢者の方々まで、あらゆる世代の方々がご利用できます。

「あっ、脳トレでもしようっと！」

そう思ったら、このアプリを少し使ってみてください。
